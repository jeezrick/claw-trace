# claw-trace v2 web

React + Vite + TypeScript + Zustand 基础骨架。

## 当前范围

- App shell
- Session list panel
- Action history panel
- Debug stream panel placeholder
- Zustand store skeleton
- API client / SSE client skeleton

## 启动

```bash
cd /root/code/claw-trace
npm run v2:web:dev
```

默认地址：`http://127.0.0.1:5174`

开发模式下 `/api` 会代理到 `http://127.0.0.1:8790`。
